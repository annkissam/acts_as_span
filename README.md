# acts\_as\_span

ActiveRecord model w/ a start\_date and an end\_date == ActsAsSpan

Treat those date spans like the objects they are!

## Getting Started

In your Gemfile:

```ruby
    gem "acts_as_span"
```

In your model:

```ruby
    class SpanRecord < ActiveRecord::Base
      acts_as_span
    end
```
