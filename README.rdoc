= acts_as_span

ActiveRecord model w/ a start_date and an end_date == ActsAsSpan

Treat those date spans like the objects they are!

== Getting Started

In your Gemfile:

    gem "acts_as_span"

In your model:

    class SpanRecord < ActiveRecord::Base
      acts_as_span
    end
